# Content/Data

Data Asset root for AshDraft tunable values (ARCHITECTURE.md 17). Avoid
hardcoding gameplay numbers; store them here.

## Folders
- `Input/`   — `UAshInputConfig` assets (InputAction <-> input tag mapping).
- `Hero/`    — hero stats / PawnData-style assets.
- `Soldier/` — soldier archetype assets.
- `Base/`    — base / stronghold settings.
- `AI/`      — AI LOD, squad, commander tuning.
- `Combat/`  — combat slot, ability set, damage tuning.
