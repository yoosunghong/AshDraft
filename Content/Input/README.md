# Content/Input

Enhanced Input assets for AshDraft (ARCHITECTURE.md 4.5).

Author these in the editor; bind them to the native input tags declared in
`Source/AshDraftCoreRuntime/Public/AshGameplayTags.h` (the `Ash.InputTag.*` tags)
via a `UAshInputConfig` Data Asset in `Content/Data/Input/`.

> These assets are generated by `Scripts/create_input_assets.py`
> (Tools > Execute Python Script). The script is idempotent.

## Initial Input Actions (IA_*)
- `IA_Move`        — Axis2D  -> `Ash.InputTag.Move`
- `IA_Look_Mouse`  — Axis2D  -> `Ash.InputTag.Look.Mouse`
- `IA_Look_Stick`  — Axis2D  -> `Ash.InputTag.Look.Stick`
- `IA_AttackBasic` — Digital -> `Ash.InputTag.Attack.Basic`
- `IA_AttackHeavy` — Digital -> `Ash.InputTag.Attack.Heavy`
- `IA_Dodge`       — Digital -> `Ash.InputTag.Dodge`

## Initial Input Mapping Context
- `IMC_AshDefault` — maps the keys/buttons (WASD, mouse XY, LMB, RMB, Space, etc.)
  to the Input Actions above.
